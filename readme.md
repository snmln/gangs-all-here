
# The gangs-all-here 
This project is intended to be an educational step by step guide of how to get started with your first minor Three.js project. Walking the devloper through how to create various objects, lighting, and adding in user input actions.

# This Branch 
This branch will go over how to change shader material of objects, add rotational motion to the object, introducing the GUI interface to chagne the position of the lighting and change the mesh wirframe setting. 

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
